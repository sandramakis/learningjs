It takes two values, can be true or false statement, yes or no, positive or negative. UNDEFINED ... It is when a variable is created but nit assigned a value

We have different ways of declaring variables, the var, let and const

The var is a depreciated way of declaring variables. let and const were introduced in 2015 in the ES6 (EcmaScript6) also known as ES2015.
let is used in block scoped functions. They are mutable, ie, its value can be re-assigned. On the other hand, const are function scoped and global scoped too. It's value are immutable, ie, it can never be changed through out the page once declared here

Observe that the third variable was written myName rather than myname or my-name.
